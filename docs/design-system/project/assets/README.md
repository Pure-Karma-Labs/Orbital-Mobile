# Assets

- **`openmoji/`** — manifest of OpenMoji glyphs used in the UI; loaded from CDN at runtime in mocks. (`assets/openmoji/README.md`.)
- **No bitmap logos** — the brief did not include a logotype or app-icon mark. The wordmark is currently `Orbital` set in Trebuchet MS Bold, blue, paired with the welcome ASCII banner. Flagged in the root README as a caveat.
- **No imagery** — Orbital is a chat product; the only product imagery is user-generated photos in threads.
